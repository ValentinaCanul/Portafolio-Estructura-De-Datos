# README ADA02

## Como ejecutar el archivo:

1- Extraiga los archivos en la misma carpeta.

2- compile el archivo: ADA02_E02.c
	Esto le creara el archivo ADA02_E02.exe

3- Ejecute ADA02_E02.exe (puede compilar y correr desde Visual Studio Code
a la vez.)

4- la terminal le pedirá el numero de clientes a atender. Elija el numero que quiera (recomendación no hacer números muy grandes, ya que puede comerse
los recursos de su computadora.

5- En la terminal debería aparecer cuando un cliente entre a una cola, y
cuando sale, junto con su tiempo de espera.

6- Al final, debería aparecer cuantos clientes fueron atendidos en cada cola, y el promedio del tiempo de espera en esa cola.
