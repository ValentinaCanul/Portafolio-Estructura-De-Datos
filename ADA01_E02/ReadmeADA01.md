# README ADA01
### Como ejecutar el archivo:

1- Extraiga los archivos en la misma carpeta.

2- Compile el archivo: ADA01_E02.c
	Esto le creara el archivo ADA01_E02.exe
	nota: si corre el archivo solo, obtendrá:
	<path_to_equations_file>, y return 1, lo 
	cual es código de error

3- Escriba sus operaciones en el archivo:

exp_infijas.txt

	nota: para su conveniencia, le hemos dejado
	uno ya llenado con las ecuaciones de ejemplo, 
	así como una 4ta ecuación que estábamos
	modificando constantemente para probar casos.

4-Desde la terminal, ya sea de Visual Studio Code o 
regular (asegurese de estar en la ruta donde esta 
guardado todo), Escriba el comando: 

.\<Nombre_del_archivo.exe> <Archivo_a_leer>

en el caso que no cambie ningún nombre de nada, el 
comando debería ser:

.\ADA01_E02.exe exp_infijas.txt

	nota: el archivo deberá estar en el mismo 
	directorio que el .exe

5- Debería poder ver los resultados en la consola, y 
también en un nuevo archivo creado (o actualizado, si 
ya esta allí) llamado:

exp_postfijas.txt
