# Proyecto ADA – Curso de Algoritmos y Estructuras de Datos Avanzadas

Este repositorio contiene las actividades desarrolladas durante el curso ADA, organizadas por carpetas numeradas.  
Cada ADA aborda una estructura o técnica distinta, implementada en C/C++ y aplicada a problemas concretos.

---

## Listado de ADAs

### ADA01 – Evaluación de expresiones con pilas  
Implementación de un analizador sintáctico que convierte expresiones infijas a postfijas usando pilas, y evalúa su resultado.

### ADA02 – Registro de estudiantes con colas  
Sistema que gestiona estudiantes mediante colas, simulando procesos de atención o asignación por orden de llegada.

### ADA03 – CRUD de películas con listas ligadas  
Aplicación que permite insertar, buscar, actualizar y eliminar películas usando listas ligadas, con validaciones de entrada.

### ADA04 – Gestión de estudiantes con árboles binarios  
Organización de estudiantes en un árbol binario de búsqueda (ABB), con reportes ordenados y visualización jerárquica.

### ADA06 – Algoritmos de ordenamiento  
Trabajo colaborativo por equipos, cada uno implementando un algoritmo clásico de ordenamiento en C (Bubble, Quick, Heap, etc.).

### ADA07 – Estructura hash con manejo de colisiones  
Desarrollo de una tabla hash para almacenar datos con colisiones controladas, usando direccionamiento abierto o listas ligadas.

## Ejecución general de los programas

1. Clonar el repositorio:  
   ```
   git clone https://github.com/usuario/proyecto-ada.git
   cd proyecto-ada
   ```
2. Entrar a la carpeta del ADA correspondiente:
  ``` 
cd ADA01
```
Compilar el archivo fuente (ejemplo con gcc):
```
gcc nombre_programa.c -o nombre_programa.exe
```
3. Ejecutar el programa
```
./nombre_programa.exe archivo_entrada.txt
```
