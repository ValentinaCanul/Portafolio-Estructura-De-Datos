# README ADA 06

### implementación de algoritmos de ordenamiento 
Este repositorio reúne el trabajo colaborativo de distintos equipos, cada uno encargado de 
implementar un algoritmo de ordenamiento en lenguje C.
La division por equipos perimitió distribuir y comparar con los otros tipos de algoritmos.
Cada uno de los equipos desarrollaba su algoritmo en una rama y una carpeta especifica.

## Algoritmos asignados por equipo
- E01 → Bubble Sort

- E02 → Quick Sort

- E03 → Cocktail Sort

- E04 → Selection Sort

- E05 → Heap Sort

- E06 → Tree Sort

- E07 → Heap Sort

- E08 → Insertion Sort

- E09 → Shell Sort

- E10 → Tournament Sort

# Estándar de lenguaje

Todos lo algoritmos son de implementarse con lenguje C. Este tipo de lenguaje de programación
permite neter un código comentado y estructurado de manera legible.

# Ejecución de los Archivos
Para compilar y ejecutar cualquier algoritmo, se recomienda seguir el mismo procedimiento:
- Clonar el repositorio y ubicar la carpeta correspondiente al equipo/algoritmo.
- Abrir la terminal en la carpeta donde se encuentra el archivo fuente (.c).
- Compilar el archivo con el compilador de C (ejemplo usando gcc):
```
gcc nombre_algoritmo.c -o nombre_algoritmo.exe
```

### ejecutar programa 
```
./nombre_algoritmo.exe
```

### Visualizacion de resultados: 
cada algoritmo mostrara en la terminal un "menú" con los
algoritmos de ordenamiento aplicando los datos definidos previamente en el código.
